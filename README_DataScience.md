## 📊 What Each Task Does

### Task 1: Iris Flower Classification
- Loads Iris dataset (150 flowers, 3 species)
- Trains 3 ML models: KNN, Decision Tree, SVM
- Compares accuracy of all 3 models
- Shows confusion matrix for best model
- **Output:** 4 PNG charts

### Task 2: Unemployment Analysis
- Analyzes unemployment data from 2018–2021
- Studies COVID-19 impact on unemployment
- Compares Rural vs Urban unemployment
- Identifies seasonal patterns
- **Output:** 4 PNG charts + CSV file

### Task 3: Car Price Prediction
- Predicts car prices using features like brand, age, km, fuel type
- Trains 3 ML models: Linear Regression, Random Forest, Gradient Boosting
- Shows feature importance analysis
- **Output:** 5 PNG charts + CSV file

---

## 🔑 Key Findings

**Task 1 — Iris Classification:**
- Best model achieves ~97% accuracy
- SVM and KNN both perform excellently
- Setosa species is easiest to classify

**Task 2 — Unemployment Analysis:**
- Pre-COVID average: ~6.0%
- COVID peak: 27.1% (April 2020)
- Recovery by 2021: ~8.5%
- Urban unemployment slightly higher than rural

**Task 3 — Car Price Prediction:**
- Horsepower and brand are top price predictors
- Random Forest gives best R² score (~0.95)
- Electric cars command premium pricing
- Every 10,000 km reduces price by ~₹15K
